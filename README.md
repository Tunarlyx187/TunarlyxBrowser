# TunarlyxBrowser
TunarlyxBrowser**, **Electron.js** tabanlı, modern ve kullanıcı odaklı bir tarayıcıdır. Şık bir girişadığın o şık tasarımı ve  arayüzü ve hızlı yapısıyla kişisel bir web deneyimi sunmayı hedefler. **Tunahan** tarafından geliştirilen bu yazılım, yenilikçi tasarımı yüksek performansla buluşturarak internette gezinmeyi daha dinamik bir hale getirir.
